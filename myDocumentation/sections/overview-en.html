  • Artistic Theory groups all the theories conceived in the art domain. It specifically focus on theories of painting. Example: Moti dell'Animo. Theory with the Aim of expressing the range of human emotion inside a painting.

  • Iconographical Motifs is the class aimed to collect all the single representations of different subjects. Example: the pictorical subject "Last Supper" has an iconographical Motifs. This motifs collects an x number of different single iconographical representations.
  • influence relation
  • Inspired by VIR: this class represent the minimum set of significative physical features without a linked meaning. Eg: the figure of Christ in Last Supper from Leonardo da Vinci is denoted by the set of colors that represent that figure inside the painting.
  • It is a significant element of an iconographical representation. it difference itself from a simple visual features due to its having an indipendence from the representation itself. Example: Last Supper has inside it the figure of Peter. Peter is a character that is independent from a single painting.
  • main character
  • painting source
  • preparatory relation
  • Primary source is the class of literary sources directly speaking about an abstract concept from the creator/theorist of it.
  • Relation describing the derivation relation between only a part of a cultural property and another. Eg: Bayonne Tapestry is a derivation of Last Supper but only for the central composition, the background is totally different.
  • scene object
  • secondary character
  • secondary source
  • Specialisation of VisualAtom, it collects the atoms identified as being a character without a link to a special meaning. Eg: the figure of Judas in Last Supper is a character, but is not linked to his identification.
  • Specialisation of VisualAtom, it represent VisualAtoms identified to be representative of composition elements of a painting. Eg: the groupings of 3 figures in Last Supper.
  • Specialisation of VisualAtom. It comprise visual Atoms aimed at showing the geometrical elements of a painting. Eg: the perspective lines of Last Supper
  • This class is an empty class for further specification. It comprises all sources about an abstract concept.
  • This class is the single iconographical representation. Each instance of this class could be present in different cultural properties and has a specific set of features that distinguish it from another one. Example: Florentine Last Supper is a way of representing Last Supper with certain temporal and location characteristics and with specific visual features.
  • This class represent a specification class for the "Concept" of Top Level Ontology. It is not thought to be directly instatiated, but only to be a collector of more defined abstract concepts as Artistic Theories, Iconographies, Iconologies interpretations and elements of paintings.
  • This class represent the iconological meaning present inside a painting or an iconography. Due to the presence of other works in that direction, further developments are not implemented.
  • tiapit: time interval
